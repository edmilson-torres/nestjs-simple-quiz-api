## Description

**Work in progress**  
[Nest](https://github.com/nestjs/nest) framework TypeScript repository API.

## Installation
Install dependencies

```bash
npm install
```
Environment variables
```bash
cp .env.example .env
```
## Running the app

Start PostgreSQL database using Docker Compose
```bash
docker compose up -d

```
Start Application
```bash
npm run start

```
